# MathPlayKids - Math Curriculum Reference

## Overview
This curriculum is based on international educational standards for mathematics education (Common Core, Cambridge, Singapore Math). Each age level builds upon the previous one.

---

## Age 5-6 (Kindergarten - Grade 1)

### Learning Objectives
- Count objects up to 20
- Recognize written numbers 1-20
- Basic shape recognition (circle, square, triangle, star, heart, diamond)
- Addition within 10 (e.g., 3 + 2 = ?)
- Subtraction within 10 (e.g., 8 - 3 = ?)

### Available Modules
| Module | Description | Example |
|--------|-------------|---------|
| Counting | Count visual objects | How many 🍎 do you see? (shows 7 apples) |
| Numbers | Recognize numerals | What number is this? (shows "15") |
| Shapes | Identify basic shapes | What shape is this? (shows circle) |
| Addition | Add within 10 | 3 + 2 = ? |
| Quiz | Mixed practice | Random from above |

## Visual Aids

### Young Children (Ages 5-8)
- Emoji objects for counting (🍎 🌟 🎈 🐻 🌸 🦋 🍪 🚗)
- Visual representations for addition/subtraction
- Large, colorful shape displays

### Older Children (Ages 9+)
- Clean numerical displays
- Mathematical notation (×, ÷, √, ^)
- Focus on mental math skills

---

## Progression Notes

1. **Scaffolding**: Each age builds on previous skills
2. **Spiral Learning**: Concepts are revisited with increasing complexity
3. **Multiple Representations**: Numbers shown visually and symbolically
4. **Immediate Feedback**: Correct/incorrect shown instantly with encouragement
